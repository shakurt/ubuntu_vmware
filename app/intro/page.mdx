# 🚀 Introduction to Ubuntu VMware Documentation

## 🤔 Why does this website exist?

This website was created as part of a university **Operating Systems Lab assignment** with the goal of providing a clear, accessible, and modern guide for installing Ubuntu on a virtual machine.

Instead of using slides or traditional handouts, this documentation is designed to be:

- 🎯 **Easy to present in class**
- 🌐 **Always available online**
- 📈 **Expandable in the future**
- 👥 **Useful for other students as well**

The project also serves as a practical exercise in using tools like **Next.js**, **Nextra**, and **GitHub** in a real-world workflow.

---

## 🎯 Who is this documentation for?

This website is designed for:

- 🎓 **University students** in Computer Engineering, IT, or related fields
- 🆕 **Beginners** who want to try Linux without installing it on their main system
- 💻 **Anyone** who needs Ubuntu for learning, practice, or development
- 👨‍🏫 **Instructors** who want a simple reference for teaching
- 📚 **New users** who want a visual or step-by-step guide

> ℹ️ **No prior Linux knowledge is required**—just basic computer skills and curiosity.

---

## 🤝 How can someone contribute or improve this documentation?

If you want to help complete, fix, or expand the content, you can contribute through **GitHub** using a **Pull Request**:

### 📝 Steps to contribute:

1. 🍴 **Fork** the repository on GitHub
2. 📥 **Clone** your fork locally
3. ✏️ **Make edits** or add new pages under the proper directory
4. 💾 **Commit** your changes with a clear message
5. 🚀 **Push** to your fork
6. 🔄 **Open a Pull Request** to the main repository

### 🎨 Your contributions could include:

- ✏️ **Fixing typos or formatting**
- 📷 **Adding screenshots**
- 📖 **Improving explanations**
- 🌍 **Writing translations**
- ➕ **Adding new sections** (e.g., troubleshooting, extra commands)
