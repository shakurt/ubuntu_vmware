# ⚙️ Installing VMware Workstation (Windows)

## 🧩 Step 1: Download the Installer

Before creating a virtual machine, you’ll first need to **download and install VMware Workstation Pro** — one of the most popular tools for running virtual machines on Windows.

You can get the installer from one of these sources:

- 🔗 [Official VMware Website](https://www.vmware.com/products/desktop-hypervisor/workstation-and-fusion)  
  Recommended for the **latest and official** version.
- 🗂️ [Alternative Mirror (Soft98)](https://soft98.ir/os/virtual-machine/1232-vmware-workstation.html)  
  Use this if the official link doesn’t work in your region.

> ⚠️ **Note:** Make sure to download the **Windows version (.exe)** of VMware Workstation.

---

## 🧭 Step 2: Verify and Prepare

After downloading:

1. Check that the file name ends with `.exe` (for example: `VMware-workstation-full-17.x.x.exe`).
2. If you downloaded a `.rar` file (from Soft98), extract it first using WinRAR or 7-Zip.
3. Close all unnecessary programs before starting the installation.
4. Ensure you have at least **5 GB of free disk space** available.

---

## ⚙️ Step 3: Run the Installer

1. Double-click the `.exe` file to start the installation wizard.
2. If prompted by Windows, click **Yes** to allow the installer to make changes.
3. Click **Next** to continue.

---

## 📜 Step 4: Accept the License Agreement

1. Read through the **End User License Agreement (EULA)** carefully.
2. Select **“I accept the terms in the license agreement”**.
3. Click **Next** to continue.

## ⚙️ Step 6: Choose Setup Options

- Most default options can be left as-is.
- (Optional) Uncheck **“Check for product updates on startup”** if you prefer not to get automatic updates.
- Click **Next** to proceed.

## ![VMware Installer Screenshot](/images/vmware/step3.png)

---

## ⚙️ Step 7: Shortcuts and User Experience Settings

- Decide whether to create **desktop or start menu shortcuts**.
- You can uncheck **“Join the VMware Customer Experience Improvement Program”** if you don’t want to participate.
- Click **Next** to continue.

## ![VMware Installer Screenshot](/images/vmware/step4.png)

## ⚡ Step 8: Start the Installation

1. Click **Install** to begin.
2. Wait for a few minutes while VMware installs on your system.
3. Once finished, click **Finish**.
4. If prompted, **restart your computer** before running VMware.

## ![VMware Installer Screenshot](/images/vmware/step5.png)

---

## 🚀 Step 9: Launch VMware Workstation

- Open **VMware Workstation Pro** from the desktop shortcut or Start menu.
- On first launch, choose one of the following:
- Enter a **valid license key**, or
- Select **“Continue with a free trial”** (30 days).

## ![VMware Installer Screenshot](/images/vmware/step6.png)

> ✅ You have now successfully installed VMware Workstation on Windows.  
> You are ready to create a virtual machine and install Ubuntu!
